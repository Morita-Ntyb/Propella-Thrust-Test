# 実験装置メインプログラム

## 概要
このリポジトリは、特定の実験装置のためのArduinoベースのメインプログラムを含んでいます。プログラムは電流、電圧、電力、温度のモニタリングと、RPMとPWMの管理を行います。

## 作成日
- 2023-11-18

## 最終更新日
- 2023-11-18

## 機能
- INA238を使用した電流、電圧、電力、温度のモニタリング
- RPMの計測とPWMの管理
- MCP3204 ADCを使用したアナログデータの読み取り
- シリアル通信を介したデータの出力

## セットアップ
### 必要なコンポーネント
- Arduinoボード
- INA238モジュール
- MCP3204 ADC
- ESC (電子スピードコントローラー)
- その他必要なセンサーと接続部品

### インストール方法
1. Arduino IDEをインストールします。
2. 本リポジトリをクローンまたはダウンロードします。
3. `include` 文に従って必要なライブラリをインストールします。

## 使用方法
1. Arduinoボードにプログラムをアップロードします。
2. シリアルモニタを開き、データの読み取りを開始します。
3. 必要に応じてプログラムをカスタマイズします。

## コード構成
- `setup()`: シリアル通信の開始、センサーとの通信の確立、初期設定の実行
- `loop()`: メインループでデータの読み取りと処理を行います。
- `manageRPMandPWM()`: RPMの計算とPWM信号の出力を行います。
- `readADC()`: ADCからのデータ読み取りを行います。
- `pulseDetected()`: RPM計測用の割り込み関数です。
